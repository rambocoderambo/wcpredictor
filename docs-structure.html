3. Prediction Engine

3.1 Score Formula (8 factors)

finalScore = EV * 0.25                          ← Expected value
            + formScore * 0.10                    ← WC form (W/D/L)
            + attackScore * 0.15                  ← Shot efficiency + possession + pens
            + defScore * 0.15                     ← Shots conceded + clean sheets + GK saves
            + disciplineScore * 0.10              ← YC + RC + fouls
            + ctxScore * 0.10                     ← Elo diff + cards + form comparison
            + lateScore * 0.10                    ← Late goals (75'+) 
            + oddsScore * 0.05                    ← Odds vs implied probability
            + divBonus                            ← Market divergence (max +5%)
            + champBonus                          ← Title probability (max +5%)

3.2 Scoring Functions v1.1

attackScore(team)

s = 0.5
  + clamp(shotEfficiency * 5, 0, 0.15)     ← goals/shot ratio
  + clamp((possession - 45) / 100, 0, 0.1)  ← possession above 45%
  + clamp(pensAwarded/matches * 0.1, 0, 0.05) ← penalties drawn
return clamp(s, 0, 1)

defScore(team)

s = 0.5
  - clamp(shotsConcededPerG / 30, 0, 0.15)   ← fewer shots conceded = better
  + clamp(cleanSheetPct * 0.3, 0, 0.1)        ← clean sheet rate
  + clamp(savesPerG / 10, 0, 0.05)            ← goalkeeper saves
return clamp(s, 0, 1)

disciplineScore(team)

s = 0.5
  - clamp(ycPerG / 3, 0, 0.1)                 ← yellow card rate
  - clamp(reds/matches * 0.15, 0, 0.1)         ← red cards
  - clamp((foulsPerG - 10) / 10, 0, 0.05)      ← fouls above 10/game
return clamp(s, 0, 1)

lateScore(team)

s = 0.5
  + clamp(lateGoalPct * 0.5, 0, 0.15)          ← % of goals after 75'
return clamp(s, 0, 1)

ctxScore(teamA, teamB)

s = 0.5
  + clamp((eloA - eloB) / 400, 0, 0.2)        ← Elo advantage
  - 0.1 if reds > 1                             ← red card penalty
  + clamp(formScore(A) - formScore(B) * 0.15, -0.1, 0.1) ← form comparison
return clamp(s, 0, 1)

3.5 Cover Probability — Poisson Monte Carlo (v1.07+)

estCover(teamA, teamB, line) — estimates chance (%) the selected team covers the Asian handicap line using 5,000 Poisson simulations per match. Replaced the old arbitrary adjustment formula entirely.

1. Expected goals (λ) per team:
     λA = (gsA / avgGoal) × (gcB / avgGoal) × avgGoal + homeAdv
     λB = (gsB / avgGoal) × (gcA / avgGoal) × avgGoal
     Uses LIVE_DATA WC goals when available, falls back to Kimi stats

2. Simulate 5,000 matches via Poisson distribution:
     gA ~ Poisson(λA), gB ~ Poisson(λB)
     goalDiff = gA - gB
     Check if goalDiff covers the handicap line

3. Cover = (fullWins × 1 + halfWins × 0.5) / 5,000

4. Blend with Kimi if fewer than 2 WC matches played

5. 50% regression to 50/50: cover = 0.5 + (cover - 0.5) × 0.5 (group) or × 0.75 (knockout)

   → Only applies when cover > 50% (favorite side). Low covers preserved unchanged.

6. Market-λ blending (v1.16): blendedCover = PoissonCover × 0.7 + (1/odds) × 0.3

   → Market-implied cover (1/odds) acts as a reality check on our Poisson model

   → Prevents overconfidence when the market disagrees with our stats

7. Exponential decay for large handicaps (v1.13+): cover × max(0.1, 1 - (absLine-1) × 0.18)

   → Only for favorites (n<0). Ensures -2 has lower cover than -1.75, -3 lower than -2.5, etc.

   → Starts at absLine=1.0 (no decay for -0.75 and below).

   → Prevents overconfidence. A 90% raw estimate becomes 70%. A 70% becomes 60%.

   → Standard calibration technique used by professional bettors.

Asian handicap line rules (simulated):

Line       | Favorite covers if       | Underdog covers if
───────────┼──────────────────────────┼────────────────────
0 (pk)     | win                      | win or draw
-0.25      | win (half on draw)       | draw or win
-0.5       | win                      | draw or win
-0.75      | win by 2+ (half by 1)    | draw or win (half by 1)
-1         | win by 2+ (push by 1)    | draw or win (push by 1)
-1.25      | win by 2+ (half by 1)    | lose by ≤1 (half by 1)
-1.5       | win by 2+                | lose by ≤1
-1.75      | win by 3+ (half by 2)    | lose by ≤2 (half by 2)

3.6 Expected Value & Fair Odds

EV = (Cover Probability × Decimal Odds) - 1
Fair Odds = 1 / Cover Probability

Reading results:

Your Odds  |  Fair Odds  |  EV        | Verdict
───────────┼─────────────┼────────────┼────────────────
2.10       | 1.85        | +13.5%    | ✅ Good value
1.85       | 1.85        | 0%        | ⚖️ Breakeven
1.60       | 1.85        | -13.5%    | ❌ Bad value

When ALL 6 bets are negative EV → ⚠ red warning banner: "Skip this match."
The tool shows this automatically — no value means no bet.

7. Prediction Engine — How It Works

7.1 Cover Probability (estCover)

Estimates the chance (%) a team covers a given Asian handicap line:

cover = Kimi match prediction blend × 40%   ← Kimi head-to-head analysis
      + Elo-based win prob × 30%            ← rating gap
      + Base stats × 30%                    ← win rate, goals, AH cover
      + Host advantage boost                ← if host nation
      - Squad age penalty                   ← if avg age > 28.5
      + Elite manager bonus                 ← Ancelotti, Deschamps, etc.
      + Handicap line adjustment            ← line vs expected goal diff
      - Upset probability factor            ← reduces confidence in known upsets

7.2 Composite Score — v1.06 Weights

Ranks all 6 bets (3 handicap lines × 2 sides) to find the best pick:

finalScore = EV × 25%           ← Expected value (cover × odds − 1)
            + formScore × 10%    ← WC form (W/D/L)
            + attackScore × 12%  ← shot efficiency + possession
            + defScore × 12%     ← shots conceded + clean sheets + GK saves
            + discScore × 8%     ← yellow/red cards + fouls
            + ctxScore × 8%      ← Elo diff + cards + form comparison
            + lateScore × 8%     ← goals after 75'
            + matchupScore × 8%  ← attack vs defense interaction (non-linear)
            + motScore × 6%      ← group stage must-win scenarios (non-linear)
            + fatigue × 3%       ← match congestion (non-linear)
            + oddsScore × 5%     ← odds vs implied probability
            + divBonus            ← market undervaluation (max +5%)
            + champBonus          ← title probability (max +5%)

7.4 Does It Favor Favorites?

Short answer: often, but not blindly.

The three main cover inputs (Kimi predictions, Elo, base stats) all correlate with team strength, so stronger teams naturally score higher. However:

  • EV (25%) is the largest factor — a -0.5 favorite at 1.20 odds has poor EV (−3%) and ranks low. A +0.5 underdog at 4.00 odds with +12% EV can outrank it.
  • When an underdog beats a favorite: poor odds for the favorite, strong underlying underdog stats (defense, discipline), late goal tendency, or known upset scenarios.
  • The upset probability factor directly reduces cover confidence for favorites in flagged upset matchups.
  • All 6 bets are ranked — the best pick isn't always the favorite side.
