# Product

## Register

product

## Users

Sports betting analysts and enthusiasts who specialize in Asian handicap odds for football/soccer. They work with match data, odds lines, and need data-driven predictions.

## Product Purpose

A minimalistic browser tool that accepts 3 Asian handicap options per World Cup match, analyzes historical team data (form, injuries, cards, rankings), calculates expected value, and recommends the single best handicap selection with confidence scoring and reasoning.

## Brand Personality

Analytical, confident, restrained. Data-forward without being cold. Clean without being sterile.

## Anti-references

Overdesigned betting sites with flashing odds tickers, loud colors, and card-heavy dashboards. This is a precision tool, not a casino lobby.

## Design Principles

1. **Data first** — every UI element earns its place by helping the user make a better decision
2. **Minimal friction** — input match data in seconds, get the prediction instantly
3. **Transparent reasoning** — show the data behind each recommendation, not just the answer
4. **Works offline** — pure static HTML with localStorage; no server needed after deploy

## Accessibility & Inclusion

Standard WCAG AA contrast. Works on desktop and mobile browsers. Reduced motion respected.
